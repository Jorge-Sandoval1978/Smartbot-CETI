# Smartbot-CETI
Smartbot CETI es un asistente automatizado diseñado para brindar información académica y servicios escolares. Su finalidad es optimizar la atención al estudiante mediante respuestas rápidas sobre carreras, requisitos, trámites, datos de contacto entre otros, reduciendo tiempos de gestión y mejorando la experiencia del usuario.

## Funcionalidades
- Responde preguntas frecuentes de estudiantes.
- Acceso inmediato a información académica y administrativa.
- Integración con WhatsApp para pruebas y demostraciones.
- Uso de Knowledge Base para almacenar datos relevantes.

## Tecnologías utilizadas
- Botpress Studio
- WhatsApp API
- Documentación en Word/PDF

## Estructura del repositorio
- `/Botpress` → Flujos y configuración del chatbot.
- `/KnowledgeBase` → Archivos de la base de conocimiento.
- `/Documentacion` → Reportes académicos y evidencias.
